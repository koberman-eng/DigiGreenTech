# Анализ конкурентов

## Анализируемые сайты

### 1. Coursera for Business
**URL:** coursera.org/business  
**Фокус:** B2B EdTech платформа

**Сильные стороны:**
- Четкая структура с акцентом на результаты
- Использование данных и метрик
- Профессиональный, корпоративный дизайн
- Сильные CTA-блоки

**Дизайн-решения:**
- Минималистичный подход
- Синий цвет как основной (доверие, профессионализм)
- Крупная типографика для заголовков
- Использование белого пространства

**Что можно взять:**
- Акцент на метриках и результатах
- Структурированная подача преимуществ
- Четкие призывы к действию

---

### 2. MasterClass for Business
**URL:** masterclass.com/business  
**Фокус:** Премиум B2B обучение

**Сильные стороны:**
- Высококачественная визуальная эстетика
- Эмоциональная связь с аудиторией
- Использование видео-контента
- Премиальное позиционирование

**Дизайн-решения:**
- Темная цветовая палитра (элегантность)
- Крупные визуальные элементы
- Фокус на контенте и экспертах
- Плавные анимации

**Что можно взять:**
- Визуальная привлекательность
- Эмоциональный подход к презентации продукта
- Качественная типографика

---

### 3. 360Learning (GreenTech аспект)
**URL:** 360learning.com  
**Фокус:** Корпоративное обучение с акцентом на вовлеченность

**Сильные стороны:**
- Современный, дружелюбный дизайн
- Акцент на инновационности
- Социальные элементы обучения
- Ясное объяснение преимуществ

**Дизайн-решения:**
- Яркие акцентные цвета
- Градиенты в ключевых местах
- Современная типографика
- Интерактивные элементы

**Что можно взять:**
- Баланс между профессионализмом и инновационностью
- Использование градиентов для современности
- Фокус на вовлеченности пользователей

## Общие выводы

### Тренды в дизайне B2B EdTech лендингов:

1. **Цветовая палитра:**
   - Фиолетовый как основной цвет (инновации, технологичность)
   - Розовый для динамичности и современности
   - Оранжевый для энергии и роста
   - Градиенты для акцентных элементов

2. **Типографика:**
   - Крупные, читаемые заголовки
   - Четкая иерархия
   - Современные sans-serif шрифты

3. **Композиция:**
   - Много белого пространства
   - Фокус на одном элементе за раз
   - Четкая структура блоков

4. **Интерактивность:**
   - Микро-анимации
   - Интерактивные элементы
   - Плавные переходы

5. **Контент:**
   - Акцент на результатах и метриках
   - Социальное доказательство (отзывы, кейсы)
   - Ясное ценностное предложение

## Рекомендации для нашего проекта

1. Объединить профессионализм (Coursera) с визуальной привлекательностью (MasterClass)
2. Использовать градиенты и геометрические абстракции для современности
3. Применить фиолетово-розовую палитру для инновационного позиционирования
4. Сделать акцент на инновационности через геометрические анимации и абстрактные формы
5. Сохранить четкую структуру и читаемость для B2B-аудитории
6. Использовать нестандартное расположение элементов (визуал слева, текст справа) для уникальности
