# Техническое задание: Лендинг для IT-стартапа

## Общая информация

**Тип проекта:** Лендинг-пейдж для IT-стартапа  
**Сфера деятельности:** EdTech/GreenTech  
**Целевая аудитория:** Инвесторы и B2B-клиенты  
**Цель:** Привлечение инвестиций и заключение B2B-партнерств

## Концепция стартапа

**Название:** DigiGreenTech  
**Миссия:** Объединение образовательных технологий с экологическими решениями для создания устойчивого будущего

**Продукт:** Платформа, которая:
- Предоставляет инновационные EdTech-решения для корпоративного обучения
- Интегрирует экологические принципы в образовательные программы
- Помогает компаниям развивать экологическое сознание сотрудников через обучение
- Использует AI и аналитику для персонализации образовательного опыта

## Целевая аудитория

### Инвесторы
- Венчурные фонды, специализирующиеся на EdTech и GreenTech
- Бизнес-ангелы, интересующиеся устойчивыми технологиями
- Корпоративные инвесторы

### B2B-клиенты
- Крупные корпорации, нуждающиеся в корпоративном обучении
- Образовательные учреждения
- Компании с фокусом на ESG (Environmental, Social, Governance)

## Основные требования к лендингу

### Блоки страницы

1. **Hero-секция**
   - Ясное ценностное предложение (Value Proposition)
   - Визуально привлекательный заголовок
   - Призыв к действию (CTA)

2. **Решения/Преимущества**
   - Ключевые преимущества продукта
   - Уникальные продающие точки (USP)
   - Визуальное представление возможностей

3. **Как это работает**
   - Пошаговое описание процесса
   - Простота и понятность интеграции

4. **Отзывы/Кейсы**
   - Отзывы клиентов или инвесторов
   - Кейсы успешного применения
   - Метрики и результаты

5. **CTA-форма**
   - Форма для связи
   - Запрос на демо/презентацию
   - Контактная информация

## Требования к дизайну

- Современный, технологичный вид
- Ощущение инновационности и надежности
- Профессиональный подход для B2B-аудитории
- Акцент на экологической тематике
- Адаптивность для всех устройств

## Технические требования

- Адаптивный дизайн (Desktop 1440px, Tablet 768px, Mobile 375px)
- Современные веб-технологии
- Быстрая загрузка
- SEO-оптимизация
