# Дизайн-концепт лендинга для IT-стартапа DigiGreenTech

## Описание проекта

Проект представляет собой дизайн-концепт лендинга для IT-стартапа в сфере EdTech/GreenTech с целевой аудиторией: инвесторы и B2B-клиенты.

**Особенности дизайна:**
- Современная фиолетово-розовая цветовая палитра
- Геометрические абстракции и анимации
- Уникальная композиция с визуалом слева и текстом справа
- Адаптивный дизайн для всех устройств

## Структура проекта

- `docs/` - Документация проекта
  - `brief.md` - Техническое задание
  - `competitors-analysis.md` - Анализ конкурентов
  - `moodboard.md` - Мудборд с трендами и референсами
  - `design-rationale.md` - Обоснование дизайн-решений
- `index.html` - HTML прототип лендинга
- `css/` - Стили
  - `styles.css` - Основные стили
  - `components.css` - Библиотека компонентов
- `presentation.md` - Презентация решения (3-5 слайдов)

## Запуск проекта

Откройте `index.html` в браузере для просмотра прототипа.

## Адаптивность

Прототип адаптирован для:
- Desktop (1440px)
- Tablet (768px)
- Mobile (375px)
